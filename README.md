# Zerofy AI

AI-powered climate intelligence platform that helps users track, predict, and reduce their carbon footprint.

## Features
- Carbon footprint calculator
- Climate Twin simulation
- AI-powered recommendations
- Sustainability dashboard
- Goal tracking

## Tech Stack
- React
- Node.js
- PostgreSQL
- AI Integration
